Aprender JavaScript de forma autodidacta tiene un problema clásico: cuando te trancas con un concepto, perder tiempo buscando respuestas en foros o documentación dispersa rompe completamente el flujo de estudio. Para resolver eso desarrollé el **Asistente Tutor de JavaScript**, una plataforma de tutoría impulsada por agentes de IA que funciona como tener a mano un equipo de expertos disponibles las 24 horas.

## La idea detrás del proyecto

La diferencia con un chatbot genérico es que esta app no tiene un solo asistente que sabe "un poco de todo". En cambio, cuenta con **6 tutores virtuales**, cada uno especializado en un área concreta de JavaScript y su ecosistema:

- **Alex** — JavaScript ES6+, closures, DOM y métodos de array
- **Tyler** — TypeScript, tipos y generics
- **Sam** — Promises, async/await y el Event Loop
- **Maya** — React hooks, Context API y optimización de rendimiento
- **Noel** — Node.js, Express, APIs y autenticación
- **Vera** — Algoritmos, Big O, recursión y programación dinámica

Cada tutor tiene una personalidad y enfoque propios, lo que hace que las conversaciones se sientan más naturales y enfocadas que hablar con un modelo genérico.

## Características que marcan la diferencia

**Sesiones paralelas**: puedes tener conversaciones activas con varios tutores al mismo tiempo. Estás estudiando closures con Alex y a la vez discutiendo promesas con Sam, sin que el contexto de una sesión interfiera con la otra.

**Historial persistente**: las conversaciones se mantienen entre sesiones. Si mañana retomas donde dejaste ayer, el tutor recuerda el hilo de lo que estaban trabajando.

**Sistema de notas integrado**: mientras conversas puedes guardar apuntes directamente vinculados a cada sesión, lo que facilita construir una base de conocimiento personal mientras aprendes.

## Stack y arquitectura

El proyecto está construido sobre una arquitectura de agentes de IA donde cada tutor es un agente independiente con su propio contexto e instrucciones de sistema. Esto permite mantener la especialización sin que un tutor "contamine" las respuestas del otro.
